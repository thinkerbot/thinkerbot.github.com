Module Configurable::IndifferentAccess
In: lib/configurable/indifferent_access.rb

Implements AGET and ASET methods that symbolize string keys, in effect producing indifferent access. IndifferentAccess is intended to extend a Hash.

Note that the indifference produced by this module is very thin indeed. Strings may still be used as keys through store/fetch, and existing string keys are not changed in any way. Nonetheless, these methods are sufficient for Configurable and ConfigHash.
