Class Configurable::Config
In: lib/configurable/config.rb
Parent: Object

Configs are used by ConfigHash to determine how to delegate read/write operations to a receiver. Configs also track metadata related to their presentation in various contexts.

Methods

[]   default   duplicable_value?   get   init   init?   inspect   new   set  

Attributes

attributes  [R]  An hash of metadata for self, often used to indicate how a config is presented in different contexts (ex on the command line, in a web form, or a desktop app).
reader  [R]  The reader method called on a receiver during get
writer  [R]  The writer method called on a receiver during set

Public Class methods

duplicable_value?(value)

Determines if the value is duplicable. Non-duplicable values include nil, true, false, Symbol, Numeric, Method, Module, and any object that does not respond to dup.
