Class Configurable::ConfigClasses::ScalarConfig
In: lib/configurable/config_classes/scalar_config.rb

ConfigClasses are used by ConfigHash to delegate get/set configs on a receiver and to map configs between user interfaces.

Attributes

default  [R]  The default config value.
key  [R]  The config key, used as a hash key for access.
metadata  [R]  A hash of information used to render self in various contexts.
name  [R]  The config name used in interfaces where only word-based names are appropriate. Names are strings consisting of only word characters.
reader  [R]  The reader method called on a receiver during get.
type  [R]  The config type for self (defaults to an ObjectType)
writer  [R]  The writer method called on a receiver during set.
