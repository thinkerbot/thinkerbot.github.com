parse(switch, value, argv)

Selects the value or the shifts a value off of argv and sets that value in config.

Parse is a hook for fancier ways of determining an option value and/or setting the value in config. Parse recieves the switch (ie long or short) mapping to self for subclasses that need it (ex the Switch class).

switches()

Returns an array of non-nil switches mapping to this option (ie [long, short]). May be overridden in subclasses.
