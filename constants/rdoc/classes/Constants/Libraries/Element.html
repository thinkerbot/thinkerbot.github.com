Class Constants::Libraries::Element
In: lib/constants/libraries/element.rb

   e.atomic_number           # => 2

   e.isotopes                # => [3, 4]
   e.abundances              # => [0.000137, 99.999863]

Gets the abundance of the isotope with the given mass number. By default, returns the abundance of the most abundant isotope.

abundance_uncertainty(n=nil)

Returns the uncertainty of the abundance of the isotope with the given mass number. By default, returns the abundance uncertainty of the most abundant isotope.

has_isotope?(n)

True if the element contains an isotope of the given mass number.

index_isotope(n)

Index of the isotope in isotopes, masses, and abundance of the given mass number.

mass(n=nil)
