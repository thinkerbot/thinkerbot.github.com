minor formatting was done on this table to make it nice for the test; the formatting consisted of condensing residue names and formula to the same line and moving composites to a separate string. ex: 

  Alanine
  C3H5NO  Ala   A   71.03712  71.08   Ala

became 

  Alanine C3H5NO  Ala   A   71.03712  71.08   Ala

Note there are minor capitalization differences in the names and abbreviations relative to those in Residue
